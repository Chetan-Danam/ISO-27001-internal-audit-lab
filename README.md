# ISO-27001-internal-audit-lab
Practical ISO 27001 internal audit checklist and evidence documentation project.
## Scenario

An internal ISO 27001 audit simulation was conducted for a fictional technology company to evaluate the effectiveness of information security controls.

The audit focused on verifying implementation of controls related to:

• Access control
• Asset management
• Incident management
• Backup procedures
• Network security
• Security awareness training

The audit checklist and evidence log in this repository demonstrate how auditors verify compliance and document audit findings during an internal ISMS audit.
